# Museo Slab

  Includes 500 regular and 500 italic fonts.
  
  http://www.fontsquirrel.com/fonts/Museo-Slab

## Installation

  Install with [component(1)](http://component.io):

    $ component install esundahl/museo-slab

## API

```css
body {
  font-family: 'Museo Slab';
}
```
